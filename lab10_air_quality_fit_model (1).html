Outline¶

Repeat exercises from Lab 9, but with the Air Quality Daily AQI dataset.

Pull the latest "Daily AQI by County" file from this link: https://aqs.epa.gov/aqsweb/airdata/download_files.html#AQI

Your target variable is CATEGORY, indicating if the air is healthy. You can focus on either the entire country, split the country in regions, or focus on just one area (ex. Utah). You can reduce noise by aggregating the data to the month of season level.

Can you predict the category based on the location and time of year?

The AQI is divided into six categories:

Air Quality Index

(AQI) Values Levels of Health Concern
0-50 Good
51-100 Moderate
101-150 Unhealthy for Sensitive Groups
151 to 200 Unhealthy
201 to 300 Very Unhealthy
301 to 500 Hazardous

def get_season(month):
    # Convert month to a cyclic representation to handle December as the next month after November
    month %= 12

    if month in [0, 1, 11]:
        return 'Winter'
    elif month in [2, 3, 4]:
        return 'Spring'
    elif month in [5, 6, 7]:
        return 'Summer'
    elif month in [8, 9, 10]:
        return 'Fall'
    else:
        return 'Unknown'

# Apply the function to create a new 'Season' column
df['Season'] = df['Month'].apply(get_season)

X = filtered_df[['CO', 'NO2', 'Ozone', 'PM10', 'PM2.5', 'Month', 'Year',
                 'Season_Fall', 'Season_Spring', 'Season_Summer', 'Season_Winter']]

filtered_df.index = pd.to_datetime(filtered_df.index)

# Perform seasonal decomposition
result = seasonal_decompose(filtered_df['AQI'], model='additive', period=12)  # Assuming yearly seasonality

# Extract seasonal component for January
seasonal_component_january = result.seasonal[result.seasonal.index.month == 1]

# Predict AQI for January based on the average seasonal component
predicted_aqi_january = filtered_df['AQI'].mean() + seasonal_component_january.mean()

# Print the predicted average AQI for January
print("Predicted average AQI for January using seasonal decomposition:", predicted_aqi_january)
